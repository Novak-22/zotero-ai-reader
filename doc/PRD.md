# Zotero PDF AI Reader Plugin - PRD

## 概述

一个Zotero插件，在PDF阅读器界面集成AI辅助阅读能力，让用户可以根据阅读的自然顺序获得LLM辅助总结。

## 核心功能

### 1. 左侧目录导航栏
- AI根据PDF内容自动生成章节/段落标题
- 支持点击标题跳转到正文对应位置
- 非Zotero内置导航栏，由插件独立实现

### 2. 右侧对话助手
- 光标选取PDF文字与LLM对话
- 累积完整上下文，支持多轮问答
- 对话历史持久化保存到本地数据库

### 3. 分段总结功能
- 按自然段落逐段调用LLM总结
- 总结内容全部输出为中文
- 总结结果显示在PDF页面旁

### 4. 多LLM支持
- OpenAI API (GPT-4, GPT-3.5)
- Claude API
- 国产大模型 (GLM, 文心, Kimi等)
- 本地LLM (Ollama)
- 用户可配置API密钥

## 用户场景

1. 用户打开一篇PDF学术论文
2. 左侧自动生成AI目录，用户可点击跳转
3. 用户按自然阅读顺序阅读，AI逐段生成中文总结
4. 选中某段文字，在右侧对话框提问AI
5. 关闭PDF后，下次打开可继续对话

## 非功能性需求

- PDF文字提取：使用Zotero内置API（文字型PDF）
- 图片/表格：v1仅提取caption文字，不做OCR
- 对话历史：SQLite本地存储
- API密钥：本地安全存储，用户可配置

## 优先级

1. **P0**: 基础框架、PDF文字提取、LLM基本对话
2. **P1**: 左侧目录生成、分段总结
3. **P2**: 对话历史持久化、多LLM支持
4. **P3**: 图片表格caption提取、国产LLM适配
